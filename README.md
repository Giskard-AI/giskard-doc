## Installation

```batch
git clone https://github.com/Giskard-AI/giskard.git
cd giskard
docker-compose up -d
```

Yes, that's all!  Then start inspecting some models at [**http://localhost:19000/**](http://localhost:19000/)****

{% hint style="info" %}
**login**: admin  **password**: admin
{% endhint %}
